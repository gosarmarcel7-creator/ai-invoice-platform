# Sift AI Frontend

Next.js is the active product surface for this repository. The App Router API routes under `frontend/src/app/api` are the source of truth for invoice upload, extraction, review, bulk actions, export, webhook dispatch, and auth email delivery.

## Local commands

```bash
npm run dev
npm run lint
npm run test
npm run build
```

## Admin console

The admin dashboard is served from `/admin` and is intended for the `a-app.docuextract.xyz` host. Set `NEXT_PUBLIC_ADMIN_HOST` and `ADMIN_BOOTSTRAP_EMAILS` in your environment before using it in production.

## Auth email setup

Password reset emails are generated by the app and sent through Mailjet. The recovery link lands on `/auth/confirm`, which establishes the Supabase recovery session and then redirects to `/update-password`.

Required env vars:

- `SUPABASE_SERVICE_ROLE_KEY` for the reset-link generation route
- `MAILJET_API_KEY`
- `MAILJET_API_SECRET`
- `MAILJET_FROM_EMAIL`
- `MAILJET_FROM_NAME`

The app's own outbound Mailjet settings live in `frontend/.env.local` and are used for invoice notification emails and password reset emails.
